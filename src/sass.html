SASS

Att börja arbeta med SASS

Jag tycker att det var lite klurigt att börja med SCSS-kod framför allt på grund av struktureringen. Jag tycker att det var svårt att sätta upp det så att det kändes som att det blev logiskt utifrån storleken på webbplatsen. Dock kan jag verkligen se nyttan av att jobba så här med större projekt då det är lättare att hitta och titta på de olika partialen/modulerna med specifik kod för just den sektionen. En nackdel tycker jag har varit att man alltsom oftast behöver köra om med npm run start för att det kraschar vid små justeringar.

Jag tyckte det var svårt i den här nedskalade varianten att veta var jag skulle lägga exempelvis mixins, i _base eller _components för att det ska bli en bra logik i det. Men sådant antar jag sätter sig mer när man väl vant sig att jobba så här. Jag tycker att det var väldigt svårt att få in en if/else-sats enligt specifikationerna, så det blev väl lite som det blev. Men det är bra att känna till att det finns olika funktioner att använda även i SCSS, men jag tänker mig att det i så fall ska användas sparsamt och i första hand använda JavaScript för det.
